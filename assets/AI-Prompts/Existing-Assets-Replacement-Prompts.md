## üîß TECHNICAL SPECIFICATIONS FOR ALL PROMPTS

### Recommended AI Tools:
1. **Midjourney**: Best for artistic, editorial style
2. **DALL-E 3**: Great for precise text integration
3. **Leonardo AI**: Good for consistent style
4. **Adobe Firefly**: Excellent for commercial use

### Output Settings:
- **Resolution**: Minimum 1024x1024 for flexibility
- **Format**: PNG with transparency for icons/logos
- **Style**: Always include "continuous line art, minimalist, Lumakara brand style"
- **Colors**: Stick to brand palette (#A3968D, #FAF8F5, #DFDACF, #262626)

### Consistency Checklist:
- [ ] Single continuous line art style
- [ ] Warm neutral color palette
- [ ] Human-centered design (hands/connection theme)
- [ ] Clean, no shading, editorial aesthetic
- [ ] Professional yet approachable tone
- [ ] Balanced negative space
- [ ] Scalable vector-friendly design
