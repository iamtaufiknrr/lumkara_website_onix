## ‚úÖ QUALITY CHECKLIST

Before finalizing any generated image:
- [ ] Single continuous line style maintained
- [ ] Exact brand colors used (#A3968D, #FAF8F5, #DFDACF, #4D403A, #262626)
- [ ] Human-centered theme (hands/connection) present
- [ ] Clean, no unnecessary elements or text overlays
- [ ] Professional yet approachable tone
- [ ] Scalable for different sizes (vector-friendly)
- [ ] Consistent with other assets in library
- [ ] No text unless absolutely essential to visual
- [ ] Transparent background for icons/logos
- [ ] High resolution for print use (minimum 1024px)
- [ ] Professional lighting and contrast
- [ ] Ultra-detailed line work quality
- [ ] Proper color context (tech=taupe, business=cacao, etc.)

## üîÑ ITERATION TIPS

### If Result is Too Complex:
Add: "simple, minimal, clean lines only"

### If Result is Too Simple:
Add: "detailed hands, expressive gestures"

### If Colors are Wrong:
Specify exact hex codes in prompt

### If Style is Inconsistent:
Always end with "Lumakara brand style, editorial aesthetic"

### If Text is Unclear:
Use "clear white sans-serif text, readable"
